Buttons

Button size

Verschillende button sizes worden gebruikt op de website van OBA wanner de layout zich aanpast aan de resolutie van het beeldscherm ( desktop, ipad, mobile).

  • Groot - btn-lg-klasse
  • Klein - btn-sm-klasse
  • Extra klein - klasse btn-xs
  
    
Code sample

    <button type="button" class="btn4 btn-xs btn-default">Button</button>
    <button type="button" class="btn4 btn-xs btn-sm">Button</button>
    <button type="button" class="btn3 btn-sm btn-default">Button</button>
    <button type="button" class="btn1 btn-lg btn-default">Button</button>

Dropdown Button

Met de vervolgkeuzelijst kan de gebruiker slechts één optie selecteren uit een lijst van minimaal 3 voorstellen (hieronder kunt u keuzerondjes gebruiken). Het gebruik ervan wordt aanbevolen voor filters of navigatiemenu's omdat dropdown-opties onmiddellijke actie hebben. Wanneer het is samengevouwen, wordt de momenteel geselecteerde optie weergegeven en wanneer het is uitgevouwen, wordt een bladerbare lijst met vooraf gedefinieerde opties weergegeven. 


  
Code sample


      <label class="dropdown">
      

      
      <div class="dd-button">dropdown
      

      
      <input type="checkbox" class="dd-input" id="test" />
      

      <ul class="dd-menu">
      

        <li>OBA
        <li>e-books & luisterboeken
        <li>Muziekweb
        <li class="divider">
        

      
        <li>Landelijk
      </ul>
      </label>

Hyperlinks Buttons

Een Hyperlink is een koppeling die verwijst naar een ander document. Dit is voornamelijk van toepassing op internetpagina's, waar gebruik wordt gemaakt van Hyperlinks om te navigeren tussen afzonderlijke pagina's.

  • agenda
  • collectie
  • leren
  • jeugd
  • service
  • vestigingen


  
Code sample


  <li>< a href="#">agenda </a> </li>
  <li><a href="#">collectie </a> </li>
  <li><a href="#">leren</a></li>
  <li><a href="#">jeugd</a></li>
  <li><a href="#">service</a></li>
  <li><a href="#">vestigingen</a>

Icon Buttons

Pictogramknoppen zijn knoppen met een pictogram erin. Een pictogramknop kan of mogelijk geen tekst. Pictogramknoppen kunnen helpen om ruimte te besparen of de kwaliteit te verbeteren mogelijkheid om knoppen snel te scannen/vinden. Oba gebruikt 4 button icons op hun webiste. 


  
Code sample
