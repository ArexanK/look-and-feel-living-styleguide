De layout

HTML heeft veel semantische elementen die verschillende elementen van de webpagina bepalen. De layout technieken die worden gebruikt om een (multi)layout te creeren in html zijn:

  • CSS frameworks
  • CCS float property
  • CSS flexbox
  • CSS grid

De OBA maakt gebruik van de techniek CSS flexbox. Hoe precies zij dit implementeren kan je hier vinden
